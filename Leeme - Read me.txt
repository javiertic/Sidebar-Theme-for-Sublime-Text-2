SUBLIME TEXT 2 LIGHT & DARK SIDEBAR

How to install:

1. Open Sublime Text 2 and go to Preferences -> Browse Packages.
2. Locate the Theme - Default folder.
3. Open the Files folder and copy its content to the Theme - Default folder.
4. In ST2 go to Preferences -> Setting - User and paste the following line of code for a light interface:
	
	"theme": "Javiertic.sublime-theme" 

or dark:

	"theme": "JavierticDark.sublime-theme"

5. Restart ST2 and you're done.
